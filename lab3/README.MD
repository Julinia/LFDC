## Further instructions shows what expressions can the parser operate
* Prefix operators as:
-25<br>
!true<br>
!false<br>
* Infix operator: 
2 + 9<br>
1 - 4<br>
* Comparison operators:
2 == 2<br>
2 !=4<br>
3 < 5<br>
5 > 7<br>
* Adding paranthesis in order to change the precedence of the operators:
(1+2) * 5<br>
* Call expressions:
add(1, 6);<br>
max(4, 8);<br>
* Let Statement: 
let x = 1 + 5; <br>
